# MotionCut Project 3 - Food Delivery App Home Screen UI

A user-friendly mobile home screen UI for a Food Delivery application[cite: 3].

## 📱 Screen Preview
![Food Delivery UI](project-3-food.png)

## 🎯 Features Designed
- **Header Section:** Delivery location ("New Delhi") and user profile icon[cite: 3].
- **Search Bar:** Rounded search bar with placeholder text[cite: 3].
- **Food Categories:** Horizontal scroll options like Pizza, Burger, Noodles, Desserts, and Indian[cite: 3].
- **Promotional Banner:** Bright colored "50% OFF" offer banner with CTA[cite: 3].
- **Popular Restaurants:** Structured cards featuring images, rating badges, and delivery times[cite: 3].
- **Bottom Navigation:** Clean bottom menu with Home, Search, Orders, Favorites, and Profile tabs[cite: 3].
